Basic Commands

Add task: $ task add Read Taskwarrior documents later

List tasks: $ task

Mark as done: $ task 2 done

Edit task: $ task 1 modify description: my edited description

Delete task: $ task 1 delete

Advanced Commands

Add priority to task: $ task add my task priority:H

Add tag and project: $ task add my task +work project:website

Add dependency to other task: $ task add my task depends:<id>

Define context and apply tag to it: $ task context define work +work

Visuals and Reports

Burndowns: $ task burndown.daily

Calendar: $ task calendar

Reports: $ task timesheet
